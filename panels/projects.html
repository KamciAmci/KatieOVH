Katie Pico 1 - Concept
Intel Atom based small form factor server. 2x10Gbe ports. 1 for no password authentication remote control and PoE power source, 1 for internet and standard authentication method remote control. 4GB/8GB/16GB/32GB LPDDR4 2400MHz DRAM module. No usb, audio, graphic ports. Preinstalled ssh no password client for control port ethernet. 100GB/500GB/1TB 7GB/s or Lan PXE support. Supported OS's: Linux ubuntu server x86, Microsoft Windows server x86
